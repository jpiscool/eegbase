# 60-second demo voiceover script

For a Loom backup recording in case Wi-Fi or BLE fails on the day. Record at 1.0× speaking pace; deck total should land at 60-70 seconds.

---

## Recording setup
- **Tab to start on:** eegbase.vercel.app/demo (welcome modal closed)
- **Click:** "▶ 60-sec tour" button (purple, top-right of demo header)
- **Tour visits in order:** Live Session → AI Insights → Brain Map → Reports → Marketing → Devices & API
- **Each tab:** ~10 seconds, advances automatically
- **Total runtime:** ~60 seconds + 5 second cold open + 5 second close = ~70 seconds

## Recording tips
- Stand up. Speak from the diaphragm.
- One take is fine. Two if the first one rambles.
- Pause briefly when each tab transition happens — gives the viewer time to register the change.
- If you stumble, don't restart the recording — just say "let me say that better" and continue. Editing out a 2-second flub looks more human than re-recording.
